L_42_Component_Vue
Создайте компонент, который будет выполнять следующее поведение: в шаблоне – должны быть две кнопки с функциями инкремента и декремента, параграф и поле ввода. Пользователь в текстовое поле вводит число, после потери фокуса данного поля это число моментально появляется ниже в значении параграфа. Значение поля очищается. При нажатии на данные кнопки число, находящееся в параграфе, должно добавлять единицу или вычитать.
